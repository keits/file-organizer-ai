# AI 智慧檔案整理器 (File Organizer AI)

這是一款專為解決數位混亂而設計的**本地優先 (Local-first)** 智慧工具。利用 AI 模型自動理解檔案內容（文字、圖片、音訊），將雜亂的資料夾（如「下載」資料夾、素材庫）自動轉化為結構化的知識庫。

> **專案目標：** 為個人開發者與中小型工作站提供一套具備隱私保護、內容感知的自動化檔案收納解決方案。

## ✨ 核心功能
- 📷 **智慧視覺命名**：利用電腦視覺模型（如 LLaVA）辨識圖片內容，自動將 `IMG_001.jpg` 轉換為描述性名稱。
- 📄 **語意分類歸檔**：透過 LLM 閱讀文件（PDF/DOCX/TXT）摘要，自動判斷所屬專案或類別。
- 🎵 **多媒體元數據提取**：自動提取影音檔案的藝術家、年份、內容描述。
- 🔒 **隱私防禦機制**：支援本地 AI 模型（Ollama / Llama.cpp），確保敏感檔案內容不離開本地設備。

## 🚀 專案定位 (補助/計畫申請亮點)
1. **技術領先性**：結合 LLM 語意理解與傳統檔案系統處理。
2. **市場需求**：解決 AI 時代資訊爆炸導致的「數位垃圾堆」問題。
3. **安全合規**：原生支援 Local LLM，解決企業對雲端隱私的疑慮。

## 🚀 快速開始 (展示模式)

為了方便評審與開發者快速體驗 AI 分類效果，我們提供了自動化展示腳本：

1. **環境準備**：
   ```bash
   pip install -r requirements.txt
   ```

2. **生成測試資料**：
   ```bash
   python make_demo.py
   ```

3. **啟動 AI 分類展示** (模擬模式，不移動實體檔案)：
   ```bash
   python main.py --dir ./demo_folder --dry-run
   ```

## 🛠️ 技術架構
- **核心語言**：Python 3.10+
- **AI 驅動**：OpenAI API / Ollama (Local)
- **檔案處理**：Watchdog (即時監控), PyMuPDF (文件分析)
